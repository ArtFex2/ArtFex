## Running the Code or .exe

Before running the code or the .exe file, ensure that you open the command prompt as an administrator. Use the following commands to install the required libraries:

**Windows:**

	pip install numpy matplotlib streamlit pillow tensorflow

**Mac:** 
	pip3 install numpy matplotlib streamlit pillow tensorflow



Running the Streamlit App


To run the Streamlit app, open the command prompt in the file location and execute the following command::

**Windows:** streamlit run streamlit.py

**Mac:** streamlit run streamlit.py



Running the .exe File


To run the .exe file, simply double-click it.

Have fun exploring the application!
